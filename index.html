    PROJETOS

    SISTEMA DE LOGIN E CADASTRO DE USUÁRIOS
    Página Web construída em HTML/CSS com tratamento e validação de dados de login/cadastro em PHP e uso de banco de dados MySQL. Esse projeto engloba o tratamento de dados feito em uma sessão de login ou cadastro de uma Página Web, direcionando e salvando os dados do cliente de forma segura no banco de dados através do sistema de cadastro para que, em sua próxima visita, o seu login já esteja salvo e pronto para ser efetuado. Esse projeto pode ser acessado por meio do GitHub.

    FERRAMENTAS E AUTOMAÇÕES ÚTEIS PARA O COTIDIANO EM PHP, Python e JavaScript
    Projetos diversos, como calculadora, cronômetro, carrossel de fotos automático, bots, além de minigames utilizando a biblioteca Pygame. Tais projetos e seus algoritmos são muito úteis para facilitar tarefas e outras diversas atividades e necessidades no dia-a-dia do usuário. Esses projetos podem ser acessados por meio do GitHub.
